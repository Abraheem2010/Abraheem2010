### ⚽ [xG Football Calculator](https://github.com/Abraheem2010/xG-Football-Calculator)
Machine learning model that predicts **Expected Goals (xG)** for football shots, trained on 2022 FIFA World Cup data.
- Gradient Boosting Classifier (scikit-learn)
- Interactive Streamlit web app with live pitch visualization
- Feature engineering: distance, angle, body part, play pattern
- `Python` `scikit-learn` `Streamlit` `Pandas` `StatsBomb`
